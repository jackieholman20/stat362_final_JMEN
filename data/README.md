## Data Setup

The dataset for this project comes from the [RSNA Pneumonia Detection Challenge](https://www.kaggle.com/competitions/rsna-pneumonia-detection-challenge/overview) on Kaggle.

**Instructions:**
1.  Download the dataset from the [Kaggle link](https://www.kaggle.com/competitions/rsna-pneumonia-detection-challenge/data).
2.  Unzip the files.
3.  Place the data in a folder named `data/` (or upload to your Google Drive if using Colab).

**Required Files:**
* `stage_2_train_images/` (Directory containing the .dcm images)
* `stage_2_detailed_class_info.csv`
* `stage_2_train_labels.csv`
